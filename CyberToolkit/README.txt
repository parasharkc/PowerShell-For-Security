Modules folder: Store .psm1 files here. These are PowerShell modules (reusable code libraries).

Scripts folder: Store .ps1 files here. These are the actual scripts that call functions from your modules.

Logs folder: To store log files generated by the toolkit.

--------------------------------------------------------------------------------
Get-FileHashValue: Generates the hash for a given file.

Monitor-FileChanges: Compares the current hash to the stored hash to detect changes.


Scan-Subnet: Performs a ping sweep on a specified subnet.


Get-LocalUserAudit: Gathers information about local user accounts (e.g., last logon, password status).
